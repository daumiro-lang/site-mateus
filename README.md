# Mateus Thessing - Site

Landing page estática para o psicólogo clínico Mateus Thessing (CRP 03/23085).

**Stack:** HTML + CSS + JS puro, zero dependências.

## Estrutura

```
site-mateus/
├── index.html       # Página única
├── styles.css       # Identidade visual (preto + lima neon + laranja)
├── script.js        # Smooth scroll + ano automático
├── assets/
│   ├── mateus.jpg   # Foto do hero
│   └── refs/        # Peças originais de referência (gitignored)
└── README.md
```

## Rodar localmente

Basta abrir `index.html` no navegador. Sem build, sem servidor.
